binary-clock.js
===============

Written in pure javascript binary clock. [Live demo](http://pasierb.github.io/binary-clock.js/).

![alt tag](http://farm3.staticflickr.com/2866/9428523630_80e4dc36c8.jpg)

Usage
-----

```
var clock = new BinaryClock(htmlElement);
```

to stop:

```
clock.stop();
```
